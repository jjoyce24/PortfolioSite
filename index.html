WDI Resources

Previous Next

WDI Resources is a MEAN stack app where past General Assembly WDI students can post links to useful tutorials and articles on programming. After posting the link users can upvote or downvote the resources which will change the order of the resources on the page. EbayScraper

Previous Next

Ebay Scraper is a tool to find ebay listings by price. The point of this project was to learn how to consume data from an API and improve on my data scraping knowledge. I used the 3taps API to get data from ebay and unfortuntley 3taps is shutting down due to legal issues with Craiglist. This will make the app very limited to use now but I learned so much about working with API's to consume data from this project, that I felt it was necessary to include it in my portfolio. Lessons I learned included: data scraping, data cleaning, script automation, and consuming JSON with an API.
